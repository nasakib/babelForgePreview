Clinical Research Abstract

Predicting PTSD Treatment Resistance
Before Intervention

Standard interventions fail in a substantial subset of PTSD patients. Current prognostic models rely on subjective clinical assessments. The Babel Forge pipeline introduces a novel machine-learning framework utilizing high-dimensional, frequency-domain neural fingerprinting to predict outcomes directly from fMRI data.

The Computational Pipeline

This section breaks down the architectural flow of the Babel Forge framework. It illustrates how raw functional neuroimaging data is transformed mathematically into a format that machine learning algorithms can analyze to predict psychiatric outcomes.

🧠

1. fMRI Acquisition

Functional MRI resting-state data is collected. The cohort consists of N=266 subjects.

➡️
🧩

2. Parcellation

The brain is divided into distinct regions using the Schaefer 200-node atlas, creating a standardized map.

➡️
🌊

3. Spectral Transform

Timeseries data transformed into the frequency domain (32 bins), yielding 6,400 discrete spatial-spectral markers.

➡️
🌳

4. Ensemble ML

A Random Forest Classifier is trained to discriminate patterns between responsive and resistant cohorts.

Babel Forge API Reference

Welcome to the Babel Forge Neural Engine API. This service exposes our 11D Random Forest model, providing real-time diagnostics for PTSD treatment resistance based on 6400-dimensional functional connectivity fingerprints.

Base URL: https://babel-forge-api-1066096962926.us-central1.run.app

1. Health Check

Verify the operational status of the Forge and ensure the Random Forest model is successfully loaded into memory.

Endpoint: /health

Method: GET

Response

{
  "status": "online",
  "model_loaded": true
}

2. Diagnose Patient

Analyze a patient's neural signature to predict their likelihood of being treatment-resistant or treatment-responsive.

Endpoint: /diagnose

Method: POST

Content-Type: application/json

Request Body

The endpoint expects a flat array of exactly 6400 float values representing the harmonized neural fingerprint.

{
  "patient_id": "sub-10294",
  "fingerprint": [0.014293, 0.013467, 0.012743, ...]
}

Response

Returns the clinical prediction along with confidence metrics and raw model probabilities.

{
  "patient_id": "sub-10294",
  "prediction_code": 1,
  "clinical_status": "RESISTANT",
  "confidence_percent": 95.52,
  "raw_probabilities": {
    "responsive_0": 0.0448,
    "resistant_1": 0.9552
  }
}

Error Responses

  • 400 Bad Request: Triggered if the fingerprint array does not contain exactly 6400 elements.
  • 503 Service Unavailable: Triggered if the neural model is currently offline or failed to initialize.
