Clinical Research Abstract

Predicting PTSD Treatment Resistance
Before Intervention

Standard interventions fail in a substantial subset of PTSD patients, and current prognostic models rely on subjective clinical assessments. The babelForge pipeline introduces a machine-learning framework that uses high-dimensional, frequency-domain neural fingerprinting to predict outcomes directly from fMRI data.

N 266 subjects D 6,400 features ACC 92.59% Atlas Schaefer 200

The Computational Pipeline

This section breaks down the architectural flow of the babelForge framework. It illustrates how raw functional neuroimaging data is transformed mathematically into a format that machine-learning algorithms can analyze to predict psychiatric outcomes.

Step 01 / acquire

fMRI Acquisition

Functional MRI resting-state data is collected across a balanced cohort of N = 266 subjects.

Step 02 / parcellate

Parcellation

The brain is partitioned using the Schaefer 200-node atlas, yielding a standardized regional map.

Step 03 / transform

Spectral Transform

Time-series data is projected to the frequency domain (32 bins), producing 6,400 spatial-spectral markers.

Step 04 / classify

Ensemble ML

A Random Forest classifier discriminates patterns between responsive and resistant cohorts.

babelForge API Reference

Welcome to the babelForge Neural Engine API. This service exposes our 11D Random Forest model, providing real-time diagnostics for PTSD treatment resistance based on 6,400-dimensional functional connectivity fingerprints.

Base URL: https://babel-forge-api-1066096962926.us-central1.run.app

GET

Health Check

Verify the operational status of the Forge and ensure the Random Forest model is successfully loaded into memory.

Endpoint: /health

Response

{
  "status": "online",
  "model_loaded": true
}
POST

Diagnose Patient

Analyze a patient's neural signature to predict their likelihood of being treatment-resistant or treatment-responsive.

Endpoint: /diagnose

Content-Type: application/json

Request Body

The endpoint expects a flat array of exactly 6,400 float values representing the harmonized neural fingerprint.

{
  "patient_id": "sub-10294",
  "fingerprint": [0.014293, 0.013467, 0.012743, ...]
}

Response

Returns the clinical prediction along with confidence metrics and raw model probabilities.

{
  "patient_id": "sub-10294",
  "prediction_code": 1,
  "clinical_status": "RESISTANT",
  "confidence_percent": 95.52,
  "raw_probabilities": {
    "responsive_0": 0.0448,
    "resistant_1": 0.9552
  }
}

Error Responses

  • 400 Bad Request — fingerprint array length ≠ 6,400.
  • 503 Service Unavailable — neural model is offline or failed to initialize.
